# Daniel Akang Single-Page Site

This repository hosts the single-page personal website draft for Daniel Akang.

Live site:

https://akang123.github.io/daniel-akang-single-page-site/

## Deployment

The site is plain HTML, CSS, and JavaScript. It is deployed with GitHub Pages from the `main` branch at the repository root.

## Update or Redeploy

1. Edit `index.html`, `assets/styles.css`, or `assets/theme.js`.
2. Commit the changes to the `main` branch.
3. GitHub Pages republishes automatically from the root of `main`.

No build step, paid hosting tier, or external package install is required.

## Local Preview

From the repository root, run:

```bash
python3 -m http.server 8097
```

Then open the local server in a browser.
